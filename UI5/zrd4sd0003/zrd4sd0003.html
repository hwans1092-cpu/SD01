* 선택한 버튼에 따라 다르게 검색!
* 일반청구

*   청구 대상 조회
    PERFORM select_bill_target.

*   조회 결과 없으면 종료
    IF gt_bill_target IS INITIAL.
*     TEXT-E01: 조회된 청구 대상이 없습니다.
      MESSAGE TEXT-e01 TYPE 'S' DISPLAY LIKE 'W'.
      RETURN.
    ENDIF.

*   청구 대상 ALV 화면 호출
    CALL SCREEN 100.

* 배치청구
  ELSEIF pa_t EQ abap_on.
*   추후 배치 구현
  ENDIF.
